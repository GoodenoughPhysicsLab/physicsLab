输入格式：  
pitch: C4, A5 ...  
rising_falling = True 时，为升调，为 False 时降调  
  
输入范围：  
C0 ~ C8  
注: C0: 24, C1: 36, C2: 48, C3: 60, ..., C8: 120
